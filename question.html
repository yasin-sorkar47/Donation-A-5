What is the Document Object Model (DOM)?

What is the Document Object Model (DOM) is interface of web pages. that represent the structure of document as a tree of object. it allows developers modify the HTML and manipulate the HTML Elements, attributes, and styles using javaScript. the DOM allows to handle the event like clicks or key presses etc.

How do you select an element from the DOM?

I can select an element from DOM Five ways.

  • 1. getElementById
  • 2. getElementsByClassName
  • 3. getElementsByTagName
  • 4. querySelector
  • 5. querySelectorAll
What is event delegation in the context of the DOM. and why is it useful?

Event delegation is a technique where i can set event parent element and it handle the child element. and it dos not make event bubbling. allow the parent to handle events from all children.

Benefits

  • 1. Improved performance.
  • 2. Handles dynamic content.
  • 3. Simpler code.
